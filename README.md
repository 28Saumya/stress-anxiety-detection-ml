# Machine Learning Models for Early Detection of Stress and Anxiety Disorders

## Overview
This project focuses on the early detection of stress and anxiety disorders using Machine Learning techniques. Multiple classification algorithms are implemented and compared to determine the most accurate model for prediction.

The system analyzes mental health-related data and predicts stress and anxiety levels based on different machine learning approaches.

---

## Objectives
- Detect stress and anxiety disorders using machine learning models
- Compare the performance of multiple classification algorithms
- Improve prediction accuracy through preprocessing and feature engineering
- Identify the best-performing prediction model

---

## Machine Learning Models Used
- Support Vector Machine (SVM)
- Logistic Regression
- Naive Bayes
- Random Forest
- XGBoost

---

## Accuracy Comparison

| Model | Accuracy |
|-------|-----------|
| SVM | 95.17% |
| XGBoost | 94.57% |
| Logistic Regression | 93.70% |
| Random Forest | 90.34% |
| Naive Bayes | 88.96% |

---

## Best Performing Model
Support Vector Machine (SVM) achieved the highest accuracy of **95.17%**, making it the best-performing model in this project.
